COVID-19 Instagram Analyzer

Social media analysis pipeline

This project involved designing and implementing a data analysis pipeline to analyze COVID-19-related behaviors and sentiments on Instagram. Our team successfully created a hashtag network that identified frequent co-occurrences of ideas among users. To gain deeper insights, we applied graph clustering techniques and centrality measures to analyze the network. Additionally, we utilized a fine-tuned BERT transformer neural network for user sentiment analysis, leveraging Google Cloud GPUs for efficient processing.
